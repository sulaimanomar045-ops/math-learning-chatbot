# وثائق المشروع

هذا المشروع هو دردشة تعليمية في مجال الرياضيات، تهدف إلى مساعدة الطلاب في فهم المفاهيم الرياضية المعقدة من خلال محادثة تفاعلية.

## الخصائص الرئيسية

- **إجابة على الأسئلة**: يوفر البرنامج إجابات فورية على الأسئلة المتعلقة بالرياضيات.
- **توجيه المستخدم**: يقدم إرشادات واضحة للطلاب حول كيفية حل المسائل الرياضية.
- **واجهة سهلة الاستخدام**: يتميز البرنامج بواجهة مستخدم بسيطة وسهلة التفاعل.

## المتطلبات

- Python 3.x
- مكتبات إضافية مثل numpy وscipy

## كيفية الاستخدام

1. قم بتنزيل المشروع من GitHub.
2. تأكد من تثبيت المتطلبات.
3. قم بتشغيل البرنامج باستخدام الأمر التالي:
   ```bash
   python chatbot.py
   ```
